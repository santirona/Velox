# Velox

A fast barcode scanner designed to streamline supermarket checkout. Users scan multiple items in their shopping cart,
and the app displays the barcodes in rapid succession for the checkout scanner to read automatically.

## Features
* Continuous vertical barcode scanning.
* Checkout mode with high-speed barcode playback.
* Adjustable barcode display speed (milliseconds).
* Bilingual support (English / Spanish).
* Works 100% offline. No database required.

## Technologies
* Kotlin
* Android Studio
* ZXing (Zebra Crossing) Barcode Scanner

## Local Build
1. Clone this repository:

   ```bash
   git clone https://github.com/santirona/velox.git
   ```
2. Open the project in Android Studio.
3. Sync the Gradle dependencies.
4. Click **Run** or build the APK.
